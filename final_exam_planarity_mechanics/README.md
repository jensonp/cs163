# Planarity Final Exam Mechanics Set

This folder collects four custom, final-exam-style questions about:

- Schnyder woods and trilinear coordinates
- extremal bounds for bipartite planar graphs
- the Auslander-Parter flap test
- primal-dual spanning trees in disconnected plane graphs

Each subfolder includes:

- the full question text
- a worked explanation
- Graphviz visuals for the key mechanism behind the answer
